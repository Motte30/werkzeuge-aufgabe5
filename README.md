## Kompilieren und Ausführen von Java-Code

##### Inhalt

1. Kompilieren

2. Ausführen

#### Kompilieren

Zum Kompilieren wird der Befehl **_javac_** benötigt. 
Dieser wird, gefolgt vom Dateinamen der zu kompilierenden Datei mit der Endung _.java_, in die Kommandozeile eingegeben.
Dabei muss die Endung (also _.java_) auf jeden Fall hingeschrieben werden.  
Durch das Drücken der **ENTER-Taste** erstellt der Kompiler eine Class-Datei, welche im nächsten Schritt ausgeführt wird.

#### Ausführen

Zum Ausführen wird der Befehl **_java_** benötigt.
Dieser wird, gefolgt vom Dateinamen der zu kompilierenden Datei mit der Endung _.class_, in die Kommandozeile eingegeben.
Dabei ist es nicht notwendig die Endung (also _.class_) hinzuschreiben.
Sie kann getrost weggelassen werden.
Durch das Drücken der **ENTER-Taste** wird der Programm-Code ausgeführt.
In unserem Fall wird "Moin Moin" ausgegeben.
